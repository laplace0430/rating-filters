# Rating Filter

Geometry Dash 온라인 레벨 검색에 Unfeatured 필터와 정확한 Featured 필터를
추가합니다.

이 모드는 Geode Index 검색 설치용이 아니라 GitHub Releases를 통한 수동 배포용
Windows Geode 모드입니다.

## 추가되는 필터

- Unfeatured
- Featured

Featured는 Epic, Legendary, Mythic을 제외한 일반 Featured 레벨만 검색합니다.
Unfeatured, Featured, Epic, Legendary, Mythic 체크박스는 서로 독립적으로
작동하며, 여러 개를 동시에 선택하면 선택한 등급들이 함께 검색됩니다.

Unfeatured와 Featured 검색에서는 제한된 범위의 서버 페이지에서 중복 없는
결과를 모아 한 페이지에 최대 10개의 레벨을 보여줍니다.

Windows용 Geometry Dash 2.2081에서 rated classic 레벨과 platformer 레벨을
지원합니다.
